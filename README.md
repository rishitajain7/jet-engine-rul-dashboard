# Jet Engine Remaining Useful Life (RUL) Dashboard

**Python | Machine Learning | Predictive Maintenance | Data Analytics**

An interactive machine learning dashboard designed to predict the **Remaining Useful Life (RUL)** of aircraft jet engines using sensor data. The system analyzes engine degradation patterns and provides actionable insights for predictive maintenance and failure prevention.

## 🎯 Overview

Predictive maintenance plays a critical role in the aviation industry by reducing unexpected failures and optimizing maintenance schedules. This project leverages machine learning techniques to estimate how many operational cycles remain before an engine is likely to fail.

The dashboard enables users to:

* Monitor engine health
* Visualize sensor behavior over time
* Predict Remaining Useful Life (RUL)
* Identify degradation trends
* Support maintenance decision-making
