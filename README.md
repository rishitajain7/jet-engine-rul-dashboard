🛠️ Jet Engine RUL Dashboard
A Streamlit-based dashboard to predict Remaining Useful Life (RUL) of jet engines using a Deep Learning model (TensorFlow).

🚀 Features
RUL prediction using trained model
Interactive dashboard (Streamlit)
Feature importance visualization
Grad-CAM interpretability

⚙️ Setup
py -3.10 -m pip install streamlit tensorflow numpy pandas matplotlib h5py

▶️ Run
py -3.10 -m streamlit run dashboard.py

Open
http://localhost:8501

📁 Files Required
dashboard.py
rul_model.h5
scaler.pkl
test_FD001.txt
RUL_FD001.txt
